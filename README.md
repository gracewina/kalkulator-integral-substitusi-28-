# Kalkulator Integral Substitusi - Gracewinna

Proyek kalkulator integral substitusi berbasis web (HTML/JS) dan aplikasi konsol (C++).

## Cara Menjalankan Versi Web:
1. Buka file `index.html` di browser Anda (Google Chrome, Edge, Firefox).
2. Masukkan nilai $a$, $b$, dan $n$.
3. Klik tombol "Hitung Integral" untuk melihat langkah-langkah dan hasil akhir.

## Cara Menjalankan Versi C++:
1. Kompilasi file `kalkulator.cpp` menggunakan compiler C++ Anda (seperti g++).
   `g++ kalkulator.cpp -o kalkulator`
2. Jalankan executable-nya:
   `./kalkulator`
